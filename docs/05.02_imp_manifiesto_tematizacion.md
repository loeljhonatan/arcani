🌌 DOCUMENTACIÓN: SISTEMA DE SINTONÍA REACTIVA (ARCANI)
Este documento describe la arquitectura de tematización dinámica basada en la Regla 90/7/3 y el Motor de Frecuencia implementado en el monorepo.
🏗️ ARQUITECTURA POR CAPAS
🟢 CAPA 1: Shared Domain (El Contrato)
Ubicación: libs/shared/domain/
Define el lenguaje común entre el Oráculo (Backend) y el Portal (Frontend).
interfaces/core/niche-theme.interface.ts: Define el objeto NicheTheme (Colores HSL, Geometría y Estado Bazar).
dtos/nichos/update-theme.dto.ts: Blindaje para el Dashboard. Valida el formato estricto H S% L%.
dtos/nichos/theming-response.dto.ts: Clase con decoradores @Expose() para permitir que la sintonía pase a través del serializador global.
🔵 CAPA 2: API Gateway (El Oráculo)
Ubicación: apps/api-gateway/
Calcula y entrega la identidad visual analizando la base de datos.
modules/core-engine/theming/services/theming.service.ts: Orquestador de base de datos. Une las tablas niche, niche_theme y niche_stock para generar el ADN visual.
modules/core-engine/theming/controllers/theming.controller.ts: Endpoint @Public() /api/theming/tune. Punto de entrada para sintonizar portales.
common/interceptors/reactive-theme.interceptor.ts: Interceptor global que inyecta automáticamente la propiedad _theme en cualquier respuesta que contenga un slug.
common/decorators/public.decorator.ts: Bypass de seguridad para rutas accesibles sin autenticación de Google.
🔴 CAPA 3: Reactive Engine (El Motor)
Ubicación: libs/core/reactive-engine/
Captura la frecuencia y la inyecta en el ADN del navegador.
services/theme-engine.service.ts: Motor basado en Angular Signals. Inyecta variables --niche- en el :root del DOM y gestiona el atributo data-arcani-state.
interceptors/sintonia.interceptor.ts: Escucha el tráfico HTTP de Angular. Si detecta un _theme, dispara la sintonía en el motor automáticamente.
🎨 CAPA 4: Frontend (La Piel)
Ubicación: apps/client-web/
Renderiza la atmósfera visual usando Tailwind 4.
src/styles.css: Procesador central. Transforma variables RAW en colores HSL, efectos de Glassmorphism y sombras neón.
src/app/app.config.ts: Registro del interceptor y configuración de capas CSS para PrimeNG.
src/app/features/nicho-view/nicho-view.component.ts: Componente de página que activa la cadena de sintonía mediante el parámetro de ruta :nichoSlug.
⚡ FLUJO DE SINTONIZACIÓN
Navegación: El usuario entra a ://arcani.com.
Petición: El NichoView solicita datos al Oráculo.
Inyección: El Interceptor de NestJS adjunta el ADN visual basado en el stock y la configuración del Admin.
Detección: El Interceptor de Angular atrapa el campo _theme antes de que llegue al componente.
Reacción: El Motor de Signals actualiza el CSS del navegador.
Inmersión: La web cambia de color con una transición de 0.8s (Fricción Positiva).
Estado del Sistema: 🟢 OPERATIVO
