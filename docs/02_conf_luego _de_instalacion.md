## Instalar Zone.js en el MONOREPO
npm install zone.js --save

```bash
npm install zone.js --save

```

## Importar Zone.js en el punto de entrada
Ve al archivo apps/client-web/src/main.ts y asegúrate de que la primera línea sea la importación de zone.js:
```bash
import './polyfills'; // Si tienes un archivo de polyfills
import 'zone.js';    // <--- AGREGA ESTA LÍNEA AL PRINCIPIO
import { bootstrapApplication } from '@angular/platform-browser';
import { appConfig } from './app/app.config';
import { AppComponent } from './app/app.component';

bootstrapApplication(AppComponent, appConfig).catch((err) =>
  console.error(err)
);


```
